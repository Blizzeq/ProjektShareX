## ​O projekcie

Strona ma za zadanie dzielenie się różnymi plikami pomiędzy użytkownikami, aby móc udostępniać pliki na stronie należy być zarejestrowanym użytkownikiem. Funkcjonalności:

- Tworzenia grup
- Zarządzanie treścią
- Przeglądanie i wyszukiwanie grup
- Powiadomienia
- Statystki
